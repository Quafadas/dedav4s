Getting Started

Getting Started

To add this library to an sbt project

libraryDependencies += "io.github.quafadas" %% "dedav4s" % "0.5.4"

To use this library in ammonite / mill

import $ivy.`io.github.quafadas::dedav4s:0.5.4`

Fire up an sbt console (or in a repl... )

import viz.PlotTargets.desktopBrowser
import viz.extensions.*
List(1,4,6,7,4,4).plotBarChart()
// res1: BarChart = BarChart(
//   mods = List(
//     viz.extensions.extensions$package$$$Lambda$16039/1159329414@61c75cad,
//     remove X axis
//   )
// )

A side effect should open a browser window, with this inside

I like a big chart... so you may also try.... 

List(1,4,6,7,4,4).plotBarChart(List(viz.Utils.fillDiv))
// res3: BarChart = BarChart(
//   mods = List(
//     viz.extensions.extensions$package$$$Lambda$16039/1159329414@6cb472cd,
//     remove X axis,
//     viz.Utils$$$Lambda$14789/414479544@6b6f5d9b
//   )
// )

If that worked, then you're ready to go! See the plot targets to understand what happened, and the examples for suggestions on how to use and extend the concepts.
